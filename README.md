# Mauzo - Point of Sale (POS) System

**Mauzo** is a modern, web-based Point of Sale (POS) system built with React, Redux Toolkit, and Tailwind CSS.  
It helps businesses track **dispatches, expenses, and seller performance**, and generate **PDF reports**.

---

## Features

- Track daily, weekly, monthly, and yearly **sales and expenses**.
- Calculate **total revenue, total expenses, and net profit**.
- Display **seller performance** with detailed statistics.
- Generate **PDF reports** for sales and dispatches.
- Responsive design, works on **desktop and mobile**.
- Easy-to-use dashboard with Tailwind CSS components.

---

## Tech Stack

- **Frontend:** React, TypeScript
- **State Management:** Redux Toolkit
- **Styling:** Tailwind CSS
- **PDF Generation:** @react-pdf/renderer
- **Date Picker:** react-datepicker
- **Icons:** react-icons
- **Version Control:** Git & GitHub

---

## Installation

1. Clone the repository:

```bash
git clone https://github.com/twende-org/mauzo.git
cd mauzo
````

2. Install dependencies:

```bash
npm install
# or
yarn install
```

3. Run the development server:

```bash
npm start
# or
yarn start
```

Open [http://localhost:3000](http://localhost:3000) to view the app in your browser.

---

## Usage

* Select the **time range** (day, week, month, year) to filter sales and expenses.
* Click **Download PDF** to export reports.
* Check the **seller performance** dashboard for insights on top sellers and revenue.
